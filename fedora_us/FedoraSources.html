Fedora Sources

Please consider using Fedora's apt and yum package because they contain enhancements for greater protection, speed and convenience.

Otherwise if you already have a non-Fedora version of apt or yum installed, be aware of the RepositoryMixingProblems. If you still wish to mix repositories, here are the entries for the Fedora stable repository as well as the Red Hat official repositories in case you don't have them from elsewhere. Please replace SERVERNAME with a real mirror server from the Fedora mirror list.

/etc/apt/sources.list entries for Fedora Core 1:

  rpm http://SERVERNAME/fedora fedora/1/i386 os updates stable
  rpm-src http://SERVERNAME/fedora fedora/1/i386 os updates stable

/etc/apt/sources.list entries for Red Hat Linux 9:

  rpm http://SERVERNAME/fedora redhat/9/i386 os updates stable
  rpm-src http://SERVERNAME/fedora redhat/9/i386 os updates stable

/etc/apt/sources.list entries for Red Hat Linux 8.0:

  rpm http://SERVERNAME/fedora redhat/8.0/i386 os updates stable
  rpm-src http://SERVERNAME/fedora redhat/8.0/i386 os updates stable

/etc/yum.conf entries for Fedora Core 1:

  [fedora-us-1]
  name=Fedora Core 1 -- Fedora US mirror
  baseurl=http://SERVERNAME/fedora/fedora/1/i386/yum/os
  [fedora-us-1-updates]
  name=Fedora Core 1 updates -- Fedora US mirror
  baseurl=http://SERVERNAME/fedora/fedora/1/i386/yum/updates
  [fedora-us-1-stable]
  name=Fedora Linux (stable) for Fedora Core 1 -- Fedora US mirror
  baseurl=http://SERVERNAME/fedora/fedora/1/i386/yum/stable
