雑 誌 (月刊誌)

日経 Linux (旧リンク http://itpro.nikkeibp.co.jp/linux/ )

当初は毎月8日発売でしたが、その後、隔月刊（偶数月）の8日発売へ変更されました。 日経BP社発行。Linuxの本格活用をねらう方々の多様な情報ニーズにこたえ，必要となる製品・技術を基礎から分かりやすく解説 し、導入例を詳しく例示するとともに，活用するためのノウハウやテクニックも掘り下げて紹介することにしているそうです。

日経 Linux のバックナンバー（既刊）のあらましは、https://info.nikkeibp.co.jp/media/LIN/?bn=mag& P=1&M=24 で知ることが可能です。雑誌類は変遷が激しく一時期のように多数刊行されなくなりましたが、日経 Linux は長期にわたり継続的に発行されています。付録DVD で入手できるインストールメディアに色々なディストリビューションを添付しているのが特色です（2020年9月8日追記）。

Linux MAGAZINE (リンク切れ)

毎月8日発売。アスキーが発行する Linux ユーザー向けの専門誌。Linux の入門を目指す初心者を対象とした記事も含まれていて、対象とする購読層が広いのが特徴です。

ただし、残念なことに2005年3月号で休刊となりました。アスキーではしばらくのあいだ過去の記事がオンラインで閲覧提供していまし たが、現在リンク切れです。ただし過去に刊行された記事は、https://web.archive.org/web /20061115025421/http://linux.ascii24.com/linux/linuxmag/ などより閲覧可能です。
