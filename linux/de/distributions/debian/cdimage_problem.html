  • Note: 3.1_r1 netinstall CD image problem - 24th December 2005
    http://cdimage.debian.org/debian-cd/3.1_r1/

    A bug has been discovered in the 3.1_r1 netinstall CD images: installable kernel images were missing. New installs from these images will most likely fail.

    The netinstall and business card CD images have been rebuilt to fix this bug; the new versions are called 3.1_r1a. We apologise for the inconvenience.
  • Note: 3.1_r0 CD image problem
    http://cdimage.debian.org/debian-cd/3.1_r0a/

    If you have already installed a system using a 3.1r0 CD/DVD image, you do not need to reinstall. Instead, simply edit /etc/apt/sources.list, look for any lines mentioning security.debian.org, change "testing" to "stable", and remove "# " from the start of the line.
