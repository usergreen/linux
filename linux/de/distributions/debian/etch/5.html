iceweasel-l10n-ja という日本語化パッケージが必要です。次のコマンドを実行してください。

 # apt-get install iceweasel-l10n-ja

Iceape Browser / Composer を単独インストールするには？

Mail や Newsgroups、Address Book を除外してインストールしたい場合は、iceape-browser を指定してしてください。提案パッケージ・推奨パッケージを全て外してインストールされます。

　iceape-gnome-support

Iceape には以下の特徴があります。

The Iceape Internet Suite consists of:
　- an Internet browser (Iceape Navigator)
　- an HTML WYSIWYG editor (Iceape Composer)
　- a Mail and News client (Iceape Mail & Newsgroups)
　- an Address Book (Iceape Address Book)
　- an IRC client (Chatzilla)
　- a Calendar (Iceape Calendar) (not officially in the suite)
　http://packages.debian.org/testing/web/iceape

Mozilla から Iceape Browser / Composer へアップグレードするには？

日本語言語パックがインストールされている場合、通常のコマンド(# apt-get upgrade)は使えません。
次のコマンドを使います。

 # apt-get dist-upgrade

ただし、日本語言語パック(mozilla-locale-ja)が強制的に削除されるので注意します。

  • iceape: Debianプロジェクト、あるいは Seamonkeyプロジェクトのいずれからも、公式日本語言語パックは未だ配布されていません(2007.01.13 現在)。

新規インストールあるいは、Sarge からのアップグレードは次のコマンドを使用します。

 # apt-get install mozilla

あるいは、

 # apt-get install iceape

ただし、2007.01.13 現在、日本語言語パッケージは未だ配布されていません。

Mozilla / Mozilla Composer 日本語版を利用したいときは？ (2007.01.13)

/etc/apt/sources.list に次の記述を追加し、

 deb http://ftp.jp.debian.org/debian/ sarge main contrib

一方、etch のapt-line を一時的にコメントします。

 #deb http://ftp.jp.debian.org/debian/ etch main contrib
 # apt-get update
 # apt-get install mozilla-browser

パッケージリストを読み込んでいます... 完了
依存関係ツリーを作成しています... 完了
提案パッケージ:
　xprt latex-xft-fonts
推奨パッケージ:
　mozilla-psm myspell-en-us myspell-dictionary
以下のパッケージが新たにインストールされます:
　mozilla-browser
（--以下省略--）

そして、ブラウザのメニューを日本語化するため日本語言語パックを追加します。

 # apt-get install mozilla-locale-ja

最後に、/etc/apt/sources.list の記述を必ず元に戻しておきます。

サードパーティのアドオン・パッケージを利用するには？

↑

debian-multimedia.org

良く知られたサードパーティ・リポジトリとして、debian-multimedia.org があります。
これを利用するには、 /etc/apt/sources.list ファイルに次の記述を追加します。

 deb http://www.debian-multimedia.org etch main
  • http://www.debian-multimedia.org/

必要に応じ以下のコマンドをその後実行します。

 # apt-get install debian-multimedia-keyring

あるいは keyring を debian-multimedia.org より入手してインストールします。

 # dpkg -i debian-multimedia-keyring_2007.02.14_all.deb

(2008-01-30 更新)
