■Debian Jessie をインストールするには?

　Debian jessie インストールガイドを見てください。 https://www.debian.org/releases/jessie/installmanual.ja.html

■古い Debian リリースからアップグレードするには?

　まずは、Debian 8 (jessie) リリースノートを見てください。 https://www.debian.org/releases/jessie/releasenotes.ja.html
　例えば、32 ビット PC を使っている場合には、次のマニュアルを見てください。
　https://www.debian.org/releases/jessie/i386/release-notes/index.ja.html (32 ビット PC 用)
　具体的には、「第4章 Debian 7 (wheezy) からのアップグレード」に詳しく書かれてある手順に従うと良いでしょう。

　リリースノートは、他にも、64 ビット PC (amd64) 用、EABI ARM (armel) 用、PowerPC 用、ard Float ABI ARM (armhf) 用、IPS (リトルエンディアン) 用、MIPS (ビッグエンディアン) 用、IBM System z 用、64 ビット PC (AArch64) 用、POWER プロセッサ用があります。

■APT の機能追加について

APT(Advanced Package Tool) は、Debian 8 Jessie のリリースと併せて、大きな変更が行なわれました。コマンドラインからの操作は以下のとおり、apt コマンドを使って簡略化できるようになりました。なお、apt-get によるコマンドも引き続きサポートされます。

apt-get update -> apt update
apt-get upgrade -> apt upgrade
apt-get dist-upgrade -> apt full-upgrade
apt-get install package -> apt install package
apt-get remove package -> apt remove package
apt-get autoremove -> apt autoremove
apt-cache search string -> apt search string
apt-cache policy package -> apt list -a package
apt-cache show package -> apt show package

Starting with Debian Jessie, some frequently used apt-get and apt-cache commands have an equivalent via the new apt binary. This means some popular commands like apt-get update, apt-get install, apt-get remove, apt-cache search, or apt-cache show now can also be called simply via apt, say apt update, apt install, apt remove, apt search, or apt show.

出典： https://www.debian.org/doc/manuals/debian-faq/ch-pkgtools.en.html#s-apt-get
