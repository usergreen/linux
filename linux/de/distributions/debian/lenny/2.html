旧安定版(Etch)からアップグレード前の事前準備

以前のリリースからアップグレードするには、
http://www.jp.debian.org/releases/lenny/i386/release-notes/ch-upgrading.ja.html
に従って、あらゆるデータや設定情報をバックアップします。

詳しくは、Debian GNU/Linux 5.0 (lenny) リリースノート (Intel x86 用)
http://www.debian.org/releases/lenny/i386/release-notes/
を見てください。

旧安定版(Etch)からアップグレードする手順は？

まずは apt と aptitude の両方または一方をアップグレードします。

「いくつかのバグ報告によれば、etch に含まれるバージョンの aptitude パッケージや apt パッケージでは、多くの場合、lenny へのアップグレードを行えません。lenny の apt では、同じようなタイミングで設定する必要がある複雑な依存関係を持つパッケージの扱いが改善されています。また、lenny の aptitude では、依存関係を満たす解決策の探索機能が改善されています。lenny への dist-upgrade 中にこれら 2 つの機能は深く関わってくるので、他のあらゆるパッケージよりも先にこれら 2 つのパッケージをアップグレードすることが必要になります。apt については、次のコマンドを実行してください。」（上記リリースノートより抜粋）

/etc/sources.list にある "etch" を　"lenny"　に変更します。その後、以下のコマンドを順次実行します。

 # apt-get update
 # apt-get install apt

なお、(aptitude をインストールしている場合) aptitude では次のように実行します:

# aptitude install aptitude
http://www.jp.debian.org/releases/lenny/i386/release-notes/ch-upgrading.ja.html#aptupgrade1st

いずれの場合にしても、次のコマンドでアップグレードの実行が完了します。

 # apt-get dist-upgrade

[関連情報]

  • Upgrade Debian Etch to Debian Lenny (http://mediakey.dk/ )
  • Upgrade Debian Etch to Debian Lenny (http://www.go2linux.org/ )

詳しくは、Debian GNU/Linux 5.0 (lenny) リリースノート (Intel x86 用)
http://www.debian.org/releases/lenny/i386/release-notes/
を見てください。

アップグレートの途中で OpenOffice.org が勝手に削除されたら？

各パッケージの依存関係が原因となって、旧バージョンの OpenOffice.org が削除されて終了することがあります。本来は、OpenOffice.org の新バージョンへ自動的に置き換わります。この場合、アップグレード完了後に OpenOffice.org をインストールすれば再び利用可能となるので心配要りません。

 # apt-get install openoffice.org

JD - 2ch ブラウザ

GTK+ based 2ch browser for Linux
JD is useful to view "2ch-style Bulletin board System" that use "bbs.cgi" and "read.cgi", it is like that we use RSS reader to read blogs. It helps you to read/write articles in such BBS.

2ch-style BBS include

  • 2 channel (http://www.2ch.net, largest BBS in Japan),
  • Shitaraba (http://rentalbbs.livedoor.com/jbbs/)
  • Machi-BBS (http://www.machi.to/) and so on.
