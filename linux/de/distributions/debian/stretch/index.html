Debian Stretch リリース情報

　Debian 9.0 (Stretch) 初版は 2017年 6月 17日 にリリースされました。
　Debian stretch に関するリリース情報 - https://www.debian.org/releases/stretch/index.ja.html

Debian Stretch をインストールするには?

　Debian stretch インストールガイドを見てください。
　-https://www.debian.org/releases/stretch/amd64/index.html.ja (64 ビット PC 用)
　-https://www.debian.org/releases/stretch/i386/index.html.ja (32 ビット PC 用)

古い Debian リリースからアップグレードするには?

　まずは、Debian 9 (stretch) リリースノートを見てください。 https://www.debian.org/releases/stretch/releasenotes.ja.html

　そして、例えば、64 ビット PC を使っている場合には、次のマニュアルを見てください。
　https://www.debian.org/releases/stretch/amd64/release-notes/index.ja.html
　なお、Debian 4.0 (etch) のリリースから、AMD64 は公式にサポートされる Debian アーキテクチャとなっています。
 　https://www.debian.org/ports/amd64/index.ja.html#status を参照してください。

　また、32 ビット PC を使っている場合には、次のマニュアルを見てください。
　https://www.debian.org/releases/stretch/i386/release-notes/index.ja.html (32 ビット PC 用)

　以下のコンピュータアーキテクチャが、このリリースでサポートされています:

    64 ビット PC (amd64)
    32 ビット PC (i386)
    EABI ARM (armel)
    Hard Float ABI ARM (armhf)
    MIPS (リトルエンディアン)
    MIPS (ビッグエンディアン)
    IBM System z
    64 ビット PC (AArch64)
    POWER プロセッサ
    64 ビット MIPS (リトルエンディアン)

インストール用の CD/DVD を自分で作るには？

　下記の Debian 公式ダウンロードサイトより、インストール用のisoイメージがダウンロード可能です。お手持ちの CD/DVD 作成ソフトウェアを使ってこれを iso イメージとして焼けば、インストール用の DVD 等を作成することが出来ます。

　https://www.debian.org/releases/stretch/debian-installer/

GNOME デスクトップ環境に関する Tips

Debian Stretch のデフォルト（標準）のデスクトップ環境は、GNOME 3 です。このデスクトップをインストールした場合の Tips は、デスク トップ環境 GNOME 3 のページを参照してください。

Stretch MATE に関する Tips

メニューの追加や編集は出来ないのです か？

Mozo は MATE 用の使いやすいメニュー編集用ツール（メニューエディタ）で、新規のエントリやメニューの追加や編集を行えます。これは、GNOME2 に標準で含まれていたメニューエディタ Alacarte を MATE 用にリネームしたものです。Debian Stretch で標準的なインストールを行った場合、これを追加インストールすることが必要です。次のコマンドでインストールします。

# apt install mozo

インストールしたら、再ログイン後にメニューエディタの利用が有効となります。メニューを右クリックして [編集] を選択するか、あるいは [システム] > [コントロールセンター] > [メイン・メニュー] を選択します。

ログイン・マネージャの設定方法が分かり ません。

Stretch MATE でインストールされる標準のデスクトップ・マネージャは、LightDM です。GNOME 3 用のデスクトップ・マネージャである GDM ではないので、注意してください。GDM とは、設定方法や振る舞いが異なっています。LightDM は 元々 Ubuntu 用に開発され、このデスクトップ・マネージャは GDM よりも動作が軽快で、GNOME 環境に依存しません。LightDM は Debian Wheezy 以降から導入されているので、Debian Stretch では充分信頼して利用できます。

具体的な設定方法は、Debian WIKI - https://wiki.debian.org/LightDM に掲載されているので、ご覧ください。

CD/DVD 作成ソフトが見つかりません。

標準的なインストールではグラフィカルな CD/DVD 作成ソフトウェアが含まれていないので、次のコマンドで Brasero を追加インストールしてくだい。なお、Debian には、Brasero 以外にもコマンドラインで操作できるものが複数用意されています。List of burn programs にCD/DVD 作成ソフトウェアの一覧が載っています。

# apt install brasero
