Binary-1 CD を使ってネットワーク・インストール（推奨）

↑

このインストールの特徴

Binary-1 CDは、 Debian Etch の標準デスクトップ環境（GNOME）の構築に必要なパッケージを含んでいます。しかし、このインストールCDだけでは、CDメディアの 容量の制限のた め、完全な日本語環境を構築できないので、インターネットに接続し、さらに約100個の Debian パッケージを自動的に追加インストールします。

ただしインストール手順は、次の項目で紹介する「テキストモード」での手順と似ていますが、ネットワークミラーを使う点に注意してくだ さい。

 11.パッケージマネージャの設定
 「ネットワークミラーを使いますか？」　＜はい＞

この方法は、完全な日本語環境を構築できる最も効率的なインストール方法であり、インストール作業に慣れていない方にも推奨できます。 インストールが完了すれば、直ちに uim-anthy による日本語入力が可能です。以下の環境も自動的に整います。

  • 日本語フォント
    VL ゴシック
    VL Pゴシック
    Kochi ゴシック
    Kochi 明朝
  • 日本語入力システム
    uim-anthy
  • CUPS 印刷システム

グラフィカルなインストールは可能ですか？

はい。インストールCDの起動時に、オプションを指定してください。

 boot: installgui

グラフィカルなインストールは、テ キストモードのインストールと基本的に同じで、見た目が異なるだけです。

インストール手順

以下の手順は、ローカル・インストール CD を使ってインストールする場合の概要(流れ)を説明したものです。（以下は GNOMEインストールCDの例）

  1. インストール CD を入手します。iso イメージからCD-R ライティングソフトを使って CD-R を作成する方法についてはここでは省略しますので、当サイト内の関連ページをご覧ください。
  2. CD-ROM ドライブにインストール CD を挿入した後、システムを再起動させます。
  3. 「Debian」というロゴとともに起動画面が表示されます。リターンキーを押してインストール作業を開始します。
  4. 「Choose Language」の画面に切り替わったら、Japaneseｰ 日本語 を選びます。
  5. 「キーボード配置の選択」では、日本(106 キー) を選びます。そうすると、すぐにハードウェアの自動検出が開始され、インストール用コンポーネントが自動的にインストールされます。
  6. ネットワークの設定をします。
  7. ディスクのパーティショニングを行います。この項目では、Fedora や KNOPPIX のハードディスク・インストール時と同様、パーティションの設定を対話式に行うことになります。必要に応じて、スワップ領域とシステ ム領域を設定します。
  8. 時間帯の設定 ハードウェアの時計は GMT に合わせていますか? ＜いいえ＞
  9. アカウントのパスワード設定
  10. Debian ベースシステムのインストールが自動的に開始され、完了すれば次の項目へ進みます。
  11. パッケージマネージャの設定
    「ネットワークミラーを使いますか？」　＜はい＞
  12. ソフトウェアの選択 (tasksel による設定) ここではデフォルトの設定のままにしておきます。（印刷サーバは自動的にデスクトップ環境に含まれます。）約695個程度 の Debian パッケージが自動的にインストールされます。 
     [*] デスクトップ環境
     [*] 標準システム
    ノートブック型（ラップトップ）にインストールする場合は、次の項目も選択すると良いでしょう。 
     [*] ラップトップ
  13. xserver-xorg の設定（必要な場合のみ）
  14. ブートローダー GRUB をインストールするか否かを選択します。これはオプションですが、Linux を単独でインストールする場合や、Windows OS インストール済みのシステムの空き領域に Linux を追加インストールする場合は、GRUB を MBR へインストールするのがふつうです。
  15. 以上インストールが完了すれば CD-ROM はドライブから自動排出されるので、その CDメディアを取り出してからリターンキーを押して作業を続行(ここではシステムを再起動)します。
  16. GNOMEデスクトップが標準で立ち上がります。この時点で、表示は全て日本語表示され、全角/半角キーの切替えで日本語入力も 行えます。
    デスクトップの解像度が適当でない場合には、# dpkg-reconfigure xserver-xorg を実行します。この設定変更を反映させる(有効とする)には、システムを再起動させる必要があります。

Debian Etch のネットワーク・インストール (Base System and Desktop)

↑

インストール手順の概要

Debian Etch インストール手順は Debian 3.1 Sarge の場合よりも相当簡略化されましたが、ベースシステム（最小構成の Debian システム）はほぼ Sarge の手順に従って進行するので、Debian Sarge のFAQ を参照してください。debian-40r0-i386-netinst.iso (ファイルサイズ約 150MB程度) を入手し、ネットワークインストールするのが最も効率的な方法です。この方法でインストールを行えば、その時点で配布されている最新の パッケージを取得で きます。

  • Debian GNU/Linux インストールガイド (日本語マニュアル)
↑

グラフィカル・インストール（GUI インストール）するとき

Debian 4.0 (Etch) では、グラフィカルなインストールをサポートします。インストールCDからの起動直後、次のコマンド・オプションを指定してください。作 業手順がその都度 表示されるので、その指示に従います。デスクトップ環境は標準で一括してインストールされます。

 boot: installgui
↑

カスタム・インストール (optional installation)

インストール時は出来るだけ最小構成で行い、必要なパッケージを後で追加する方法は、問題発生時の特定を容易にするだけでなく、ハード ディスクの占 有容量を大幅に節約できます。それには、tasksel の「デスクトップ環境」と「標準システム (Standard System)」のチェックを外すことです。以下「最小構成でインストールするとき」の項目を見てください。

  • バー ジョン 4.0 (Etch) では、インストール時の日本語 True Type フォントに「さざなみ」(ttf-sazanami-mincho, ttf-sazanami-gothic) を指定してください。これ以外のフォントをインストールした場合、文字化けする恐れが高くなります。

最小構成でインストールした後の環境設定（まとめ）

最小構成でインストールを行った場合は、必要なパッケージを追加し、運用可能なようにデスクトップ環境を構築する必要があります。以下 はその例です。

  1. 最小構成のインストールが完了すると、自動的に再起動した後、画面はログイン待ちの状態になります。
  2. まず、ユーザーでログインします。
  3. デスクトップ環境をインストールします。
    次のコマンドを実行します。関連パッケージは芋づる式にインストールされます（gnome や kde、Xfce4 はその目的で用意された「メタパッケージ」です）。 
     # LANG=C apt-get install gnome gdm xserver-xorg x-window-system
    (LANG=C は万一のための文字化け対策です。)

    上記コマンドの実行では、パッケージの総ダウンロードサイズが約260 MBに過ぎません。tasksel で「デスクトップ環境」選択時は約 650 MBですので、これだけで大幅な作業時間短縮につながります。KDE環境を選択したい場合は gnome を kde に、Xfce環境を選択したい場合は gnome を Xfce4 に置き換えて実行します。Xセッションマネージャには、gdm の替わりにシンプルな xdm の選択が可能です。
  4. 上記の追加インストールが完了するのを待ちます。
  5. 追加インストール完了後、X の起動テストをします。
    一度ユーザーに戻り、コマンドラインより X の起動テストを行います。 
     # exit
     $ startx
  6. X の起動テストが OK であれば、次の手順に進みます。
    もし Xが正常に起動できないときは、ビデオカードの設定が適切でないのが原因かもしれませんので、ビデオカードの再設定を実行してください。 
     # dpkg-reconfigure xserver-xorg
  7. システムの再起動をしてください。 
     # reboot
    gdm をインストールしたときは、グラフィカルなセッションよりログインすることができます。gdm をインストールしなかったときは、コマンドラインより X を起動させます。
  • なお、Etch のインストールでは、通常最低限の日本語フォントも併せてインストールされるようになっています。より品質の高いフォントを利用した い場合は、フリーフォントを参照してください。次のコマンドでも、日本語 true type フォントがインストールできます。 
     # apt-get install ttf-sazanami-gothic ttf-sazanami-mincho
    あるいは、 
     # apt-get install ttf-vlgothic

以上の作業で標準的なデスクトップ環境が整います。引き続いて、次項の日本語入力環境の設定へ進みます。
