Composer をインストールできますか？

Composer のメニューを日本語化したい。

上記項目の手順でインストールされる Composer は英語版ですから、メニューは日本語になりませんが、「日本語言語パック」を追加インストールすることで対処できます。日本語言語パックは、SeaMonkey? 本家プロジェクトより入手する必要があります。

  • 対応するバージョン：seamonkey-2.0.11.ja.langpack.xpi
  • 入手先：ftp://ftp.mozilla.org/pub/mozilla.org/seamonkey/releases/

なお、ダウンロードした言語パッケージは、Composer のメニューから Edit > Preferences > Advanced > Software Installation > Add-on Manager > Install を選択してインストールします。

Iceweasel 巻き込んで落ちまくってまともに観られません。(2013.3.29)

本家版である Firefox とは仕様が全く同じではありません。微妙な相違から特有の問題が発生していると考えられます。
flash-player に関わるこの不具合は、当方においても確認しています。

Chromium ウェブブラウザを一時的に代用するのが良いでしょう。Chromium ウェブブラウザは、Debian の公式版が入手できます。
# apt-get install chromium-browser chromium-browser-l10n
