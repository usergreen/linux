Sylpheed 2.0 インストール (for Sarge)

安定版向けには、sylpheed 1.0 が用意されています。sylpheed 2.0 の安定版用リリースは、2006.05.23 現在ありません。

↑

Sylpheed 2.0 (testing)

公式な testingリポジトリよりインストールし利用することは可能です。sylpheed 2.x が、3.1 Sarge でも動作します。testingリポジトリを /etc/apt/sources.list に追加し、

 deb http://ftp.jp.debian.org/debian/ testing main

インストールを実行します。

 # apt-get update
 # apt-get install sylpheed

インストール後、/etc/apt/sources.list の設定は必ず元に戻します。

Sylpheed 2.0 (unofficial)

Sylpheed 2.0 へ移行する（非公式パッケージ利用の場合）には、以下の手順で設定します。

  • 非公式パッケージ　sylpheed-2.0 for Sarge (apt line) 
     deb http://everybody.good-day.net/~ikuya/debian/sarge/sylpheed-1.9/ ./
