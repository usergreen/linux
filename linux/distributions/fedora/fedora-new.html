Fedora の日本語入力環境について

ibus-anthy が標準で採用されています。辞書は kasumi を使うことができます。
https://docs.fedoraproject.org/ja/fedora/f34/release-notes/desktop/I18n/

日本語入力へ切り替えるには？

Fedora は最近のバージョンで入力切替に使用する標準キーを変更したようです。インストール直後は、半角/全角キーが効きません。
英語入力から日本語入力への切り替えは、Windows(Super)キー＋スペースキーです。一度日本語入力を有効にすると、その後の切り替えは半角/ 全角キー、または Ctrl + スペースが使えます。これはおかしな仕様だとは思いますが、担当開発者が英語圏のせいだと思います。英語環境では 日本語-Anthy を追加するだけなので設定の重複は発生しません。

この変な仕様を修正するには、メニュー > システム > ユーザ向け > 入力メソッドのセレクタ の項目で、重複する日本語のうちの一方「日本語」を削除することで解決します。「日本語-Anthy」は残します。この設定画面は、パネルの IBus アイ コンをクリックし、入力メソッドの項目を開くことでも同様に行えます。
参照サイト: 「日本語(ja)なる謎の物があったので削除」 http://linux.ikoinoba.net/index.php?UID=1359558980

ログイン直後は英数入力にできませんか？

Fedora 34 の日本語入力設定は、デスクトップにログインした直後から入力切り替えなしですぐ日本語が入力できます。この標準設定を変更するには、パネルのIBusア イコンをクリックし、「設定」項目で初期値をひらがなから英数へと変更してください。これで次回ログイン時より変更が有効になります。

好きな日本語フォントを追加したい

たとえば VL ゴシックを追加するには、次のコマンドで行えます。* 印はワイルドカードで、これによってゴシック体とプロポーショナル・ゴシック体の2つがインストールされます。
# dnf -y install vlgothic*

インストールに必要なハードウェア環境は？

標準の x86_64 ワークステーションをインストールするためには、次のシステム環境が推奨されています。
2GHz quad core プロセッサー
4GB システムメモリ
20GB のドライブの空き容量
出典: https://docs.fedoraproject.org/ja/fedora/f34/release-notes/welcome/Hardware_Overview/

デスクトップ環境の選択について

Software Selection でデスクトップ環境が選択可能です。 (Fedora Workstation の標準は GNOME Desktop です)
  • Cinnamon Desktop
  • KDE Plasma Workkspaces
  • LXDE Desktop
  • LXQT Desktop
  • MATE Desktop
  • Sugar Desktop Environment
  • Xfce Desktop
なお、Live.iso の標準は GNOME です。ISO イメージを利用してインストールする場合、次のようなコマンドで他のデスクトップ環境を追加でインストールできます。
# dnf groupinstall "KDE Plasma Workspaces"
# dnf install @mate-desktop-environment
出典: https://docs.fedoraproject.org/en-US/fedora/f34/install-guide/install/Installing_Using_Anaconda/#sect-installation-gui-software-selection

dnf とは何ですか？

これは従来の yum に代わる新しいパッケージマネージャです。使い方は以前とほとんど同じです。dnf という名前は dandified yum から来ています。粧し込んだ高機能な yum ということなのでしょう。

グラフィカルインストーラが動きません

指示された処理の解決に時間が掛かっていると考えられます。長時間無反応な場合はウィンドウ右上の × をクリックして強制終了することができます。コマンドラインから dnf を使う方が快適に処理することが可能です。

root アカウントがありません

ライブ ISO のインストーラは root アカウントを作成せず、sudo を使用します。そのため、root アカウントを利用するにはパスワードの作成が必要です。
$ sudo passwd root

Firefox ブラウザの日本語文字が汚いのですが

標準フォントが noto sans cjk に指定済みですが、システム設定が適切でないと考えられます。ブラウザの既定言語を英語 (en_US) に変更してブラウザを再起動すれば、非常に綺麗に表示されるようになります。おそらくは英語環境でしか検証されていないのでしょう。ただし、ブラウザのイ ンターフェース (UI) は英語になってしまいます。

別の方法としては、上記で紹介した VL ゴシックをインストールして、ブラウザの既定フォントに指定するやり方があります。VL ゴシックであれば適正化されているらしく、綺麗に表示されます。
