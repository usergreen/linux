Fedora Linux について

Fedora というのは、Linux ディストリビューションの１つです。現時点 (2005.03.18) では、Fedora Project から Fedora Core 1 / 2 / 3 / 4 という3つのバージョンがリリース済みです。

  • トップページ(総合案内ページ)： Fedora Project.org http://fedoraproject.org/
  • 公式メインサイト： Fedora Project http://fedora.redhat.com/
  • 関連サイト(派生プロジェクト)
    • Fedora Extras Home Page http://www.fedora.us/
    • Fedora Legacy Project http://www.fedoralegacy.org/

Intel(R) 845G Chipset 搭載パソコンが起動しない問題

インテル製 845G チップセットを搭載したマザーボードを使用するデスクトップ・パソコン、あるいはこれを内蔵したノートブック・パソコンが特定のカーネルのもとではシステ ムの起動に失敗するという不具合が見つかっています。具体的な症状は、起動画面が表示された直後ディスプレイがブラックアウトし、システムが停止するとい う状態になります。

この不具合は、Fedora Core 1 の特定のカーネルで確認できています。Fedora ではパッケージのアップデートが頻繁に行われていますので、ある日突然起動しなくなった場合には、まずこの問題を疑った方が良いと思われます。

これまでにリリースされたカーネルのうち、当方で正常な起動が確認できたバージョンを載せておきます。

  • Linux version 2.4.22-1.2199.nptl
  • Linux version 2.4.22-1.2197.nptl
  • Linux version 2.4.22-1.2179.nptl
  • Linux version 2.4.22-1.2115.nptl

カーネルのアップデートには、慎重を期した方が良いでしょう。

Fedora Core 1 EOL とその後の対応について

Fedora プロジェクトの運営に関する委員会は、Fedora Core 3 Test 2 が2004年9月20日付でリリースされたので、同日を以って Fedora Core 1 のメンテナンス(サポート)をFedora Legacy プロジェクトに移管しました。

移管後は、Fedora Legacy プロジェクト からサポートが行われる予定となっています。

  • FC1 Support Started! http://fedoralegacy.org/

    The Fedora Project officially ended support for Fedora Core 1 (FC1) on September 20th, 2004. Fedora Legacy has now picked up support for FC1.

  • Fedora Core 1 Transferred to Fedora Legacy (fedora-announce-list@redhat.com Mon, 20 Sep 2004)

    With the release of Fedora Core 3 Test 2, the Fedora Steering Committee would like to announce the transfer of Fedora Core 1 to the Fedora Legacy Project.

    This means that updates for Fedora Core 1 will no longer be posted at http://download.fedora.redhat.com/.

    For more information on the Fedora Legacy Project, or if you wish to join the team please see http://fedoralegacy.org/.

Up2date を利用する

/etc/sysconfig/rhn/sources に次の記述を追加します。

 # Fedora Legacy FC1 os repository
 yum os http://download.fedoralegacy.org/fedora/1/os/i386/
 # Fedora Legacy FC1 updates repository
 yum updates http://download.fedoralegacy.org/fedora/1/updates/i386/
 # Fedora Legacy FC1 legacy-utils repository
 yum legacy-utils http://download.fedoralegacy.org/fedora/1/legacy-utils/i386/
↑

APT を利用する

/etc/apt/sources に次の記述を追加します。

 rpm http://download.fedoralegacy.org/apt fedora/1/i386 os updates legacy-utils

YUM を利用する

/etc/yum.conf に次の記述を追加します。

If you're using yum, please add the following blocks to your/etc/yum.conf (these work for all supported releases of Fedora Core Linux):

 [base]
 name=Fedora Core Linux $releasever base
 baseurl=http://download.fedoralegacy.org/fedora/$releasever/os/$basearch 
 
 [updates]
 name=Fedora Core Linux $releasever updates
 baseurl=http://download.fedoralegacy.org/fedora/$releasever/updates/$basearch 
 
 [legacy-utils]
 name=Fedora Legacy utilities for Fedora Core $releasever
 baseurl=http://download.fedoralegacy.org/fedora/$releasever/legacy-utils/$basearch
