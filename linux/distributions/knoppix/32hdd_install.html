KNOPPIX 3.2 Hard Disk Installation

KNOPPIX をハードディスクにインストールする手順です。

このページでは、knx-hdinstall.ja を使ってハードディスクにインストールする具体的な手順を説明します。

ただし、このページの手順は、主として KNOPPIX 3.2 以前のバージョンを対象とします。（バージョン 3.3 でもこのコマンドはサポートされていますが、今後メンテナンスを行なわない方針とされているようです。）


以下、ハードディスクに未使用の空き領域がある場合を基本に説明します。

1. まず始めに、KNOPPIX を　CD　から起動させます。起動後、KNOPPIX ＞　Root Shell を開き、次のコマンドを入力します。

 # knx-hdinstall.ja

リターンキーを押すと、日本語によるスクリプト（インストーラー）が立ち上がりますので、マウスで　OK　をクリックして開始します。これからイン ストールしようとするハードディスクの一覧が表示されるので、確認のうえ　OK　を選択します。

2. パーティション・ツール「cfdisk」が起動するので、OK　ボタンを選択して画面を進めていきます。

3. 領域テーブルを作成するか尋ねてきますので、「Y」を入力して、新規作成画面に進みます。

4. 基本領域を選択して、Enter　キーを押します。

  • Windows がインストールしてある場合でこれを削除しても構わないときには、その領域を選択し、
    →　キーで削除の項目にカーソルを合わせて　Enter　キーで削除します。
  • 使っているパソコンにすでに Windows がインストールしてある場合で、これを残しておきたいが他に利用できる空き領域がないときは、別途、空き領域を作成する必要があるので、作業を中止してく ださい。
  • 他の　Linux　ディストリビューションをインストール済みで、これを保持する必要がない（削除してもよい）場合には、この領域を削除して 空き領域を作成することが可能です。

5. Linux 用の　swap パーティションを作成する場合には、任意にその分だけ差し引いて基本領域の数値を入力します。
例えば、Linux 用に 10GB の領域を充てることができる場合で、swap パーティションを 512MB　に設定したいのなら、10GB-512MB=9.488MB という計算になるので、正確には 9488 という数値ですが、おおよその数値 9400 を入力しても問題ありません。

6. この基本領域から起動可能とするために、「ブート可」を選択して　Enter　キーを押します。

7. 同じく、swap パーティション用の基本領域を追加作成します。

8. 次に、→ キーで、「FS タイプ」を選択し、Enter キーを押します。一覧が表示されますが、さらに　Enter キーを押します。

9. 一覧の中から、「82 linux swap」を選択し、Enter キーを押します。

10. 「書き込み」を選択し、確認のメッセージが表示されたら、「yes」と入力します。これで、領域テーブルに書き込みが開始されます。

11. 書き込みが済めば、「終了」を選択して Enter キーを押します。

12. swap パーディションを利用するか尋ねてくるので、「Yes」を選択します。続いて確認画面が現れるので、そのまま進んでいきます。

13. ファイルシステムの選択画面に表示されます。選択は次の３つのうちから行います。

 ext2
 ext3
 reiserfs

14. フォーマットを開始する確認画面が表示されるので、OK　を押します。

15. 続いて、KNOPPIX CD からハードディスクへファイルのコピーが行われます。

16. ファイルのコピーが終了すると、確認画面が表示されるので、OK　と押します。

17. 起動時の各種サービスの設定画面となります。
mail サーバー、secure shell サーバー、samba サーバー、print サーバーの選択ができるので、使っている環境にしたがって、任意に設定します。

18. 起動時にグラフィカルログインを行うか聞いてくるので、希望するときは　Yes を選択します。

19. 「ホスト名」を入力します。デフォルトでは、「box」となっていますが、好きな名前を付けることができます。

20. 「DHCP のブロードキャスト」を使うかどうかを尋ねてきます。ブロードバンドルーターを使ってネットワークに接続している場合には、Yes と答えておきます。No と答えた場合には、ネットワークの設定を行う画面になります。

21. root ユーザーや、追加のユーザーアカウントの設定を画面にしたがって行います。

22. ブートローダー（Lilo）をインストールするかどうかを設定します。ここでは　Yes と答えます。（No と答えた場合には、別途 KNOPPIX の起動ディスクが無いと、次回から通常の手順では起動できなくなってしまうので、十分注意します。）

23. boot フロッピー（起動用ディスク）の作成をメッセージにしたがって行います。

24. これで、KNOPPIX のハードディスクへのインストール作業は完了です。

Root Shell を終了させるために exit と入力してリターンキーを押します。次に、ログオフしてください。しばらくして CD が自動的に排出されるので、忘れずに取り出します。

25. Linux を再起動させます。いくつかの個人設定を求められた後、画面に「Debian」という文字が大きく表示され、KNOPPIX を使い始めることができます。
