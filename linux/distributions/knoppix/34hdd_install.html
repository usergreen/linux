KNOPPIX 3.4 Hard Disk Installation

KNOPPIX のハードディスクへのインストール方法 -V3.4 (20040629)日本語版向け。

ここでは、 KNOPPIX 日本語版(20040629) をハードディスクにインストールする手順を説明します。このバージョンには、さらに改良されたインストーラー Knoppix Installer V03-11 が採用されています。

(注意) 以下は KNOPPIX V3.4(20040629)版 日本語版をハードディスクにインストールした場合(例)です。これよりも以前にリリースされたバージョンをインストールしたいのであれば、以前のインス トール方法のページ 3.3HddInstall あるいは 3.2HddInstall を見てください。パーティションを切り直す必要がある場合も、以前のページの関係部分の説明を参照してください。

1.デスクトップの左下にある　「K アイコン」から、＞　KNOPPIX ＞　Root Shell　と選択します。
root のコンソール画面が表示されます。

2.次のコマンドを入力して、リターンキーを押します。

 # knoppix-installer

3.「このスクリプトは、Knoppix を Hard Disk へインストールするものです。注意：このバージョンは、とても初期段階でまだかなり開発中のものです・・・。」というメッセージが表示されるので、確認の うえ OK を押します。

4.次の一覧が表示されます。

 1.インストールの設定　新しい設定を作成します
 2.インストールの開始　インストールを開始します
 3.パーティション設定　ハードディスクを分割します
 4.設定の読み込み　　　既存の設定を読み込みます
 5.設定の保存　　　　　設定を保存します
 6.終了　　　　　　　　このプログラムを終了

5.「1.インストールの設定」を選ぶと次の3つの選択肢が表示されます。

 beginner: マルチユーザーシステムと自動認識 (より良い方法)
 Knoppix： CD 起動のような Knoppix-System
 debian：　Debian 形式のシステム（以前のインストーラの様に）

この画面では、「beginner: マルチユーザーシステムと自動認識 (より良い方法)」を選択しましょう。これは Linux 初心者向けに新しく追加されたインストール方法です。インストール中の自動設定・自動認識に優れているので、作業がさらに簡略化できるようになります。

6.次に、パーティションの選択画面になります。最も適当と思われるパーティションが自動的に選択されるので、確認のうえ作業を進めてください。

7.次に、システム名の設定をします。ここでは適当な名前を付けておきます。（例： name surname）

この名前は、初期ユーザー作成に使用されます。入力が済めば、Next を押します。

8.ユーザー名を入力します。好きなユーザー名（例：roroot）を入れて、Next を押します。

9.ユーザーのパスワードを設定し入力します。確認のため２回入力してから、Next を押します。

10.管理者(root)用のパスワードを設定し入力します。確認のため２回入力してから、Next を押します。

11.ホスト名を設定します。そのパソコンの名前になります。（例：box）

12.ブートローダー (lilo) を入れる場所を指定します。

多くの場合、Master Boot Record を選ぶことになります。

13.自動的に最初の一覧画面に戻ります。この中から、「2.インストールの開始」を選びます。

14.「これらの設定で実行してもよろしいですか？」というメッセージが表示されます。

これでよいのであれば、 Next を押します。

この画面は、インストール直前の最終確認の画面です。インストールの設定に関する変更を希望するときは、 この画面をキャンセルし
元に戻って再度設定し直してください。 Next ボタンを押すと、取り消すことはできません。

15./dev/hda* の初期化が開始され、続いて CD からハードディスクへのコピーが行なわれます。そのあいだの経過状況は、進捗バーで表示されます。

ここでの一連の所要時間は、およそ２０分以上かかります。やがてインストールは、自動的に完了します。完 了した時点で、ブートディスク作成の画面に切り替わります。

16.ブートディスク（起動用ディスク）を作成をするかどうか尋ねられます。

作成を希望する場合は、 フロッピーディスクを挿入して Yes を押します。

17.「インストール成功」の画面が出ます。

「ハードディスクへ Knoppix を入れることに成功しました。」というメッセージが表示されます。

18.システムを終了させます。 Root Shell は、 exit で閉じ、ログアウトします。

正常に終了できれば CD が自動的に排出されるので、忘れずに取り出してからリターンキーを押します。これでパソコンの電源が切れます。

19.再び、パソコンの電源を入れます。途中でログイン画面が表示されるので、登録したパスワードを入力します。

20.パソコンが完全に起動したら、 KNOPPIX を使い始めることができます。
