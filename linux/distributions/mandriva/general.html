Mandriva Linux とは何ですか?

Mandriva Linux は、フランスに本部を置く Mandriva 社によって開発された商用のデスクトップ Linux ディストリビューションです。リテール版の売上高では世界一を誇るディストリビューションです。Mandriva は、"Man-dree-vah"　あるいは「マンドリーバ」と読みます。

このディストリビューションが初めてリリースされたのは 1998年7月のことであり、当時は Red Hat Linux (バージョン 5.1) と KDE (バージョン 1.0) をベースとしたものでしたが、その後多くの開発が加えられた結果、独自ディストリビューションへと発展しました。従来、 Mandrakelinux あるいは Mandrake Linux と（開発当初は Linux Mandrake とも）表記されていましたが、商標権の問題とその後のブ ラジルの Conectiva 社との合併のため、 2005 年4月7日に正式発表された社名変更 によって、 Mandriva Linux となりました。製品紹介は、こちら にあります。Mandriva Linux - フリー百科事典『ウィキペディア』Wikipedia, the free encyclopedia (日本語版はこちら) も、Mandriva Linux の概要を知るには適切なページです。

無償配布のダウンロードエディション (Mandriva Linux Download Edition) が入手可能です。最新版の Mandriva Linux 2006 Edition については、2005年11月14日、無償の FTP版が公開されました。（※Mandriva Limited Edition 2005 以降リリース方針が変更され、今後の製品の名称は年号ベースとなりました。）

このページでは以下、ダウンロードエディションの利用を中心として紹介します。

Mandriva Cooker とは何ですか？

次期公式版としてリリースされる予定の開発中の版は「Cooker」と呼ばれます。
将来のリリースに向けて改良を加えたり検証を行ったりする目的で、この「Cooker」は一般公開されます。多数の開発者やテスターが Cooker の開発に参加しコミュニティを形成しています。

  • http://www.mandriva.com/en/community/contribute/join
  • How To Participate in the Mandriva Cooker Project
