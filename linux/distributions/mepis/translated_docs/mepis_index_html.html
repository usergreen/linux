Mepis Linux とは?

MEPIS Linux は、他の Linux ディストリビューションとは相違点があります。MEPIS Linux のCDは、３種類のCDが１つになっています。つまり：

  • ハードディスクにインストールしなくても起動する ライブCD
  • ハードディスクにインストールするためのCD
  • Linux の全機能が利用可能な、リカバリー用途や修復作業用のCD

MEPIS Linux は、本来３枚のCDに収められる程の内容を２枚のCDまたはISOイメージとして配布されています。これは、このライブCD 兼インストールCDの内容が圧縮して記録されているからです。

ソフトウェアの特徴

MEPIS Linux は、標準的な Debian GNU/Linux から派生したディストリビューションです。標準的な Debian GNU/Linux とは互換性があります。

ライブ/インストール用CDは、1400を越える実行プログラムを含む、900以上のパッケージを備えています。その中には、次のもの があります。

  • Mozilla ブラウザー 1.7.2 を含む X Windows システム 4.3.0 が稼働する KDE 3.1。
  • MPEG ビデオ形式や MP3 形式に対応する X マルチメディア・システム(XMMS) および、Ogg Vorbis オーディオ・プレーヤー。
  • PPP や WiFi? に対応したインターネット接続用のソフトウェア。
  • 画像編集用のソフトウェア GIMP バージョン 2
  • サン・マイクロシステムズ社のオフィス・スイート StarOffice? のオープンソース版 OpenOffice 1.1.2　
  • ソフトウェア開発用の多くのプログラム言語や開発環境 (kdevelop など)
↑

システムの要件

  • インテル製ペンティアム・クラス互換のCPU (AMD製も利用可能)
  • 128 MB 以上のメモリ
  • 起動可能な CD-ROM/DVD ドライブ、あるいは起動可能な FD ドライブと CD-ROM/DVD ドライブ
  • ハードディスクにインストールするためには、2 GB 以上の空き容量が必要

Mepis Linux の利用を開始するには?

使用しているコンピューターが、CDから起動可能であることを確かめてください。BIOS で起動順序を変更するか、あるいは CD から起動するようにマニュアルで設定変更する必要があるかもしれません。CDから直接起動できない場合には、次の手順で起動用FDを用い れば CDから起動可能になるかもしれません。(以下の「起動用フロッピーディスクの作成方法」参照)

まず始めに、CDをドライブに挿入してから、コンピューターを再起動します。この時、CDから起動するように設定しておきます。やがて MEPIS という画面が表示されると、その約10秒後には MEPIS Linux が自動的に起動します。

もし正常に起動しない時には、起動時のオプションを選択したうえで、再度CDからの起動を試してください。
