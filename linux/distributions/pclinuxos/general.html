  • GNOME デスクトップ環境のインストール 
     # apt-get install task-gnome
    あるいは、 
     # apt-get install task-gnome-minimal
  • Xfce デスクトップ環境のインストール 
     # apt-get install task-xfce

HDD へのインストール

デスクトップのインストール用アイコンを選択します。このアイコンは、/usr/sbin/draklive-install　への ショートカット です（root権限を必要とします）。
コマンドラインからは、次のコマンドを実行します。

 # draklive-install

drakelive-install
Live installer for PCLinuxOS
This tool allows to install PCLinuxOS from a running live CD.
