日本語環境 (For Japanese Environment)

  • 日本語環境 (For Japanese Environment)
    • パッケージの追加インストール
    • ユーザーのロケール変更方法（ユーザーレベルでの変更）
    • 日本語版リマスターCD を入手する

以下の手順どおり、パッケージの追加インストールとシステムのロケール変更を行います。

↑

パッケージの追加インストール

  • KDE デスクトップの場合 
     # apt-get update 
     # apt-get install scim scim-anthy scim-qtimm scim-input-pad fonts-ttf-japanese
  • GNOME / Xfce デスクトップの場合 
     # apt-get install scim scim-anthy scim-input-pad fonts-ttf-japanese locales-ja
     # apt-get install mozilla-firefox-ja openoffice.org openoffice.org-l10n-ja
↑

ユーザーのロケール変更方法（ユーザーレベルでの変更）

  1. Panel > System > Configuration > Other > LocaleDrake? (User)
  2. Set the language as Japanese.
  3. Log-off & log-in
