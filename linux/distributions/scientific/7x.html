LiveCD/DVD から起動できないのですが？

まず、BIOS の起動順次の設定を確認する必要があります。LiveCD/DVD の ISO イメージを CD/DVD または USB で作成した場合、そのメディアが最初に起動するように設定されている必要があります。お使いのパソコンの説明書で確認してください。

また、それが UEFI をサポート（採用）するコンピュータである場合には BIOS の起動設定が正しくてもおそらく起動しないでしょう。 Scientific Linux の LivCD ISO イメージには EFI ブートエリアが含まれていないので、これを作成する必要があります。これに対処するには BIOS 設定に関する十分な知識が要りま す。

同じ RHEL のクローンである CentOS LiveCD は SL7 とは仕様が異なり、EFI ブートエリアも含まれています。LiveCD を利用したいときはこれを利用する方が簡単に解決します。

SL バージョン 8 が見つかりません

Scientific Linux 8 はリリースされないことなりました。次期メジャーリリースでは CentOS 8 を採用することが正式に決定されたからです。CentOS 8 についても、事情によって１年後にはプロジェクトが終了します。詳しくは CentOS 7 のページを見てください。

CentOS プロジェクト終了後の対応については 2020年12月20日現在、検討中であることが CentOS 8 Linux@CERN https://linux.web.cern.ch/centos8/ というページで告知されています。
