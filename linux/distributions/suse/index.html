SUSE LINUX について

SUSE LINUX は、ドイツの SUSE 社が開発し販売している Linux ディストリビューションのひとつです。日本では Red Hat Linux が広く知られていますが、ヨーロッパでは最も普及しているディストリビューションとなっています。最近、アメリカ Novell 社の買収によって、日本での販売は Novell 社(日本法人) があたるようになりました。無料版をダウンロードし、インストールすることができます。

下記 SUSE LINUX 9.1 インストールガイドでは、「SUSE LINUX 9.1には、有料の製品版と無料でご利用になれるFTP版があります。機能的にはほぼ同じですが、FTP版ではライセンス上の理由などで含まれていないも のもあります。また、FTP版にはサポートは含まれていません。」という説明が掲載されています。有料の製品版と無料で利用できるFTP版の2つが存在す るという扱いは、既に販売が終了した Red Hat Linux と似ています。

CD から起動できるバージョン (Bootable CD) は提供されていますか。

はい。LiveCD という名称で、無償で配布が行われています。このバージョンは、コンピューターのハードディスクにインストールしないで Linux を起動させることが可能です。国内では 1CD Linux としてすでに人気のある KNOPPIX 同様、Linux が気軽に運用できます。

上で紹介しているダウンロードサイトから入手できます。

  • 最新のバージョン 
     SUSE-10.0-LiveDVD.iso
  • 過去のバージョン 
     LiveCD-9.1-01.iso
     SuSE-9.0-LiveEval-i386-Int-RC1.iso

このライブCDの使い方についてはマニュアルが出ているので、その中の英語版を引用しておきます。( 9.0-LiveEval? の場合もほぼ同じです。)

How to Use the SUSE LINUX 9.1 Live CD

1. Download the ISO file LiveCD-9.1.iso from the specified Internet address and save it on your hard disk. For this you will need about 700 MB of storage space.

2. Use a burning application of your choice to burn the ISO file on a blank CD.

3. Insert the CD in your CD or DVD drive and reboot the machine. In contrast to a system installed on the hard disk, the live CD consumes a lot of memory. Therefore, at least 256 MB RAM are needed for the smooth operation of the live CD.

4. SUSE LINUX 9.1 will boot directly from the live CD without modifying your hard disk.

引用先：ftp://ftp.suse.com/pub/suse/i386/live-cd-9.1/README.txt

LiveCD-9.1-01.iso is the first update of the Live CD/SUSE LINUX 9.1,
published on May 6th 2004.

Bootable CD で日本語が使えないのですが。

LiveCD-9.1-01 では、標準の言語が、英語とドイツ語となっています。そのため、通常の方法で起動させた場合、日本語入力が出来ません。ただし、Gothic フォントがインストールされているため、付属のブラウザーを用いて日本語サイトを閲覧することは問題なく可能です。

SuSE-9.0-LiveEval? では、標準で日本語入力が出来ます。どうして、最新版の LiveCD-9.1-01 で日本語入力が機能しないように設定されているのかは不明です。日本語で文書の作成を行いたい方は、SuSE-9.0-LiveEval? という旧バージョンを利用してください。
