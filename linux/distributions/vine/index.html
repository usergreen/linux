CDイメージはどれをダウンロードすればいいんでしょう?

http://vinelinux.org/getvine.html#ViaFTP の各サーバの下記のファイルです。

  • Vine Linux 2.6 
     Vine-2.6/CDIMAGE/Vine26r4-i386.iso
  • Vine Linux 3.2 
     Vine-3.2/CDIMAGE/Vine32-i386.iso

PowerPC の Mac の場合は i386 の代わりに ppc を使います。なお、Vine Linux 3.2 の PPC 版についてはリリースされていませんので、Vine Linux 3.1 を使用してください。
SRPMS.iso は OS のインストールには不要です。

CD の作り方についてはインストール用のCDを作りたいんですが?参 照のこと。

APT の使い方について教えてください

APT - Advanced Packaging Tool (Vine Linuxオンラインマニュアル) など参照。

基本となるのは次のコマンドです。

コマンド 解説
# apt-get update 最新のリストに更新する
# apt-get upgrade 更新されたリストに従って、パッケージの更新を行う
# apt-get dist-upgrade 依存関係を解消しながら、OS 全体の更新を行う
# apt-get install パッケージ名 個別に指定してインストールする
# apt-get remove パッケージ名 個別に指定して削除（アンインストール）する
# apt-get clean 一時保存されているキャッシュファイルを削除する
$ apt-cache search キーワード キーワードからパッケージを検索する
$ apt-cache show パッケージ名 パッケージの説明をみる
$ apt-cache list-nodeps インストール済みのパッケージのうち、他のパッケージから要求されていないものを表示する

以前のバージョンから 2.6r4 へ乗り換える必要がありますか。

2.6r4 の RELEASE-NOTES.ja にその注意書きがあるので、引用しておきます。

2.6r1/r3 を既にお使いの方は再インストールやアップグレードは基本的に必要ありません。ほぼ全てが errata として提供されていますので apt-get 等で常に最新を保っていれば問題はありません。また一部のバグフィックスも 2.6r4 に対して apt-get upgrade することで修正されます。

デスクトップでマウスの右クリックが出来ません。

Vine Linux 2.6 以前のバージョンでは、インストール直後のデフォルトの状態で、 GNOME ファイルマネージャー」が有効となっていないからです。Fedora や KNOPPIX など他のディストリビューションを使っている方は戸惑うかもしれません。

メイン・メニュー(足マークのアイコン)　＞プログラム　＞ファイルマネージャーと選択すれば、ファイルマネージャーが起動し、デスクトップには予 め登録されているアイコンが表示されます。同時に、右クリックも機能するようになります。

Vineはイイ!んだけど パッケージが古くさくて xxxが使えない/用意されていないのでどうしましょう?

  • いくつか解決策
    • 自分でいれる
      • src.rpm を他のディストリ（または VineSeed）から持って来て作り直す?
      • /usr/local以下にソースからいれる
    • Seedに移行
      • apt-line を VineSeed? に向けて、次のコマンドを実行する 
         # apt-get update
         # apt-get install apt
         # apt-get dist-upgrade 2>&1 | tee upgrade.log
        使用しているシェルが tcsh の場合は 2>&1 | の代わりに |& としてください。
      • 失敗した場合は upgrade.log を確認する

文書作成ソフト OpenOffice.org (OOo) を使いたい。

  • Vine2.6 にはパッケージとしてはありませんが、OpenOffice.org の公式サイト からダウンロードしてインストールすれば問題なく使えます。具体的なインストール方法は公式サイトにありますが、次の関連項目にも出ています。
    • Open Office.org の最新日本語版 (1.1.4) を使えるようにしたい。
      Vine Seed にもあります。apt-get コマンドでインストールしてください。
      ただし、Vine Seed は基本的には常に unstable という位置づけになるので、重要なシステムでは使わないことを推奨しています。
  • [Vine 3.0 でAPTを利用する場合の一例] 
     # apt-get update
     # apt-get install openoffice.org openoffice.org-i18n
     パッケージリストを読みこんでいます... 完了
     依存情報ツリーを作成しています... 完了
     以下の追加パッケージがインストールされます:
      openoffice.org-libs
     以下のパッケージが新たにインストールされます:
      openoffice.org openoffice.org-i18n openoffice.org-libs
     アップグレード: 0 個, 新規インストール: 3 個, 削除: 0 個, 保留: 20 個
     172MB のアーカイブを取得する必要があります。
     展開後に 691MB のディスク容量が追加消費されます。
     続行しますか? [Y/n]y
    インストール後、コマンドラインから OpenOffice.org を起動するには、$ oowriter のようにします。
    使用する言語が日本語または英語だけなら openoffice.org-i18n はインストールしなくても構いません。

無線LANカードが利用できないのですが。

Vine Linux 2.6 では、付属のグラフィカルなネットワーク設定ツールに不具合があります。
Vine Linux 2.6 での Melco WLI-PCM-L11, WLI-PCM-L11G 設定例を載せておきます。

付属のグラフィカルなネットワーク設定ツールを使用すると、この Melco のネットワークカードを有効にできないので、直接設定ファイルを次のように編集する必要があります。

/etc/sysconfig/network-scripts/ifcfg-eth1 ファイルを適当なエディターで開き、次の記述を冒頭に追加します。

 DEVICE=eth1
 BOOTPROTO=dhcp
 ESSID=******
 KEY=s:******
 ONBOOT=no
  • ESSID= の ******の部分は、実際の設定値を調べて入力します。
  • KEY=s: ******の箇所は、未設定なら不要です。16進数で設定している場合は、s: を付けません。

設定後、/sbin/ifup eth1 で起動し、ネットワークに接続できます。

  • この設定方法は、Vine Linux 2.93 (3.0 Beta) でも利用できます。
↑

無線LANカードが利用できないのですが。(その2)

Vine Linux 3.0 の場合です。[アプリケーション] ＞ [システム・ツール] ＞ [ネットワークの管理] よりネットワークの設定を行えます。2.6 と比較して設定ツールは改善されたようですが、WEP KEY を有効にしている場合にはその設定項目が存在しないため自動接続ができず、設定が正常に完了できません。
そこで、 /etc/sysconfig/network-scripts/ifcfg-eth1 ファイルを適当なエディターで開くと、次のように(一例)記述されているでしょうから、

 BOOTPROTO=dhcp
 ONBOOT=yes
 USERCTL=yes
 NAME="無線 LAN カード"
 DEVICE=eth0
 ESSID=**********
 PEERDNS=yes

さらに次の一行を追加してください。これでたいていは接続できるようになるでしょう。

 KEY=s:*************

起動時の OS 選択画面で linux と linux2.2up という二つの項目が表示されます。

Vine Linux 2.5/2.6 では2種類のカーネルがインストールされます。

linux を選ぶと起動するのは 2.4 系のカーネルです。特に問題が起きていないならこれを使いましょう。linux2.2up は古い 2.2 系のカーネルを起動します。

間違って Vine Linux を二つインストールしてしまったわけではないので安心しましょう。

↑

インストール作業はマウスを使わなくても行えますか。

はい、もちろんグラフィカルなインストールを行えます。

最近多く用いられている USB マウスは、使用環境によってはインストール時、機能しないことがあります。キーボードからの入力が可能であれば、接続されているマウスを一切使用しないで 作業をすすめることが可能です。その場合の基本的な操作は、次の3つだけです。

  • 設定項目間の移動は、Tab キーで行います。連続して押打すると、最初の項目へ一巡させることができます。
  • 各項目内での選択は、←↑↓→矢印キーを使います。
  • いずれの場合でも、選択の確定は Enter キーを押すことで行います。

なお、インストール作業が無事完了すれば、システムの再起動後、接続されている USB マウスは利用可能な状態になっているはずです。

webmin がインストールできないんですけど…。

Vine 3.1 では、/etc/apt/sources.list の apt-line に extras を追加すればインストールできます。

 # apt-get update ; apt-get install webmin

なお、/etc/vine-release を書き換える必要はありません。

新しいウィンドウ・マネージャーをインストールしたんだけど、wdmが認識しません。

/etc/X11/wdm/wdmReconfig*に項目を追加します。
たとえばxfce4なら次の項目を付け加えてください。

 # entry for xfce4
 rc_word=xfce4-session  #ここにウィンドウ・マネージャを立ち上げる命令を書く。
 XFCE_PATH=""
 for rc_dir in $path; do
   if test -x $rc_dir/$rc_word; then
      XFCE_PATH=$rc_dir/$rc_word
      WMLIST="$WMLIST:xfce4"
      setupWM xfce4 ${XFCE_PATH}
      break
   fi
 done

プログラムのソースを見たい。

 $ apt-get source pakagename

カレント・ディレクトリに pakagename-version-release.src.rpmができる。

 $ rpm -ivh pakagename-version-release.src.rpm

$HOME/rpm/SOURCES にソースが展開される。
Vine 3.0 以降では src.rpm の取得後、自動的に展開されるようです。
