インストール直後に発生する問題について

YUM が自動的にインストールされないので、インストールディスク 4 より手動で追加する必要があります。ディスク 4 をドライブに挿入すれば自動マウントされるので、端末エミュレータより次のコマンドを実行してください。

 # rpm -ivh /mnt/cdrom/WhiteBox/RPMS/yum-2.2.0-0.fc3.noarch.rpm

"Yum won't get installed as part of the base system so stick CD4 in and install it
before using either it or Up2date Network."

  • http://beau.org/pipermail/whitebox-announce/2005-May/000077.html

（環境によっては、自動マウントされないことがあるようです。また、CD-ROMのディレクトリも異なることがあります。）

 # mount /media/cdrom/
 # rpm -ivh /media/cdrom/WhiteBox/RPMS/yum-2.2.0-0.fc3.noarch.rpm
 # umount /media/cdrom/
