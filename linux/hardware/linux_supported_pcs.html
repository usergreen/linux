Linux 作動確認パソコンについて

Linux_supported_PCs - Last updated: Nov. 28, 2020

Linux をコンピューターにインストールするに当たって、無線 LAN がサポートされているかどうかは重要です。最近のノートパソコンはたいてい無線 LAN が内蔵されているため、これを直接利用できるのであれば大変便利です。

Linux サポート情報の公開サイト

次のページで Linux の作動確認情報がメーカー別に公表されています。また、Linuxディストリビューションのインストールやセットアップの解説を行っているサイトもあります。メーカーから販売されたすべての機種を網羅しているわけではないので、それぞれの製品仕様を確認したうえ で役立ててください。
  • Linux on Laptops -- https://linux-laptop.net/
  • Supported Hardware /Linux journal -- https://www.linuxjournal.com/supportedhardware 
  • Debian Project による作動確認情報  -- https://wiki.debian.org/InstallingDebianOn

  •  InstallingDebianOn -- メーカー別の各機種毎の Debianのインストールと設定方法
    "DebianOn is an effort to document how to install, configure and use Debian on some specific hardware."
    https://wiki.debian.org/InstallingDebianOn

  • Complete Guide to Installing Linux on Chromebook -- グーグル・クロームブックへのLinuxディストリビューションのインストール
    https://itsfoss.com/install-linux-chromebook/

実際に確認を行なった PC

ノートブック PC（法人向けの安価なPCで個人使用でも購入可能なものを含む）のうち、Debian/GNU Linux のインストールを行って正常に作動したのでお知らせします。内蔵無線 LAN も作動しました。無線 LAN を内蔵しないタイプは無線 LAN カードで検証しました。なお、2014年4月時点では、無線 LAN (wireless lan) には、Atheros AR9462 が多くのベンダーで採用されていました。
当方で実際に作動確認したPCを以下掲載します。

- Fujitsu FMV LIFEBOOK AH52/DA (*1)
- Lenovo (IBM) ThinkPad R61e (*2)
- Lenovo ThinkPad L530 (*3)
- NEC VersaPro VR-E (*4)
- TOSHIBA dynabook Satellite B453 (*5)
- TOSHIBA dynabook B65/H (*6,*7)
- Epson Direct -- Endeavor NT2600 (*8)
- SONY VAIO -- PCG-F75/BP (*9)
