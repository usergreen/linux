Current modern machines for practical usage -- Last updated: 3/26/2021


Linux をインストールするのに適当だと思われるマシーン（PC本体）を参考までに紹介します。選考基準は日常の作業で十分実用的でかつ比較的安価な製品のうち、 Linuxディストリビューションが無難に作動すると推定されるものです。また、日本国内で容易に入手可能な機種に限ります。
システム構成を見て購入・選択時の参考として活用してください。ただし Linux の作動保証は一切できません。
