システム情報を表示させるには？ 

  •  # dmidecode
    CPU の詳細な仕様やメモリー、マザーボードのベンダー、メモリーの仕様、さらには場合によっては本体の製造番号さえも掴むことが可能です。
    このコマンドの詳しい使い方は、次の man ページで知ることができます。 
     $ man dmidecode
  • グラフィック・チップの対応状況
    Intel GMA software support for Linux (en:Wikipedia) の記事が良くまとめられているので参考になります。
  • X バックライト調節ユーティリティ (xbacklight)
    X.org バージョン7.3 では、xbacklight というバックライト調節ユーティリティが同梱されるようになりました。xbacklight は Debian Lenny や Mandriva 2008 でも提供されています。コマンドラインより、次のようにして使用します。 
     $ xbacklight -set 50
