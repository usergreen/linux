イーサネットコンバータ（無線 LAN中継機）を使う

このタイプの機器はルーターからの無線を変換して、パソコンの有線用端子である LANコネクター（RJ-45 と呼ばれる規格のポート）への接続を可能にします。無線LANコンバータ、ワイヤレスLANコンバータ、Wi-Fiコンバータなどの名称でも販売されてい ます。複数のメーカーから製品が発売されていて、BUFFALO社製 WEX-G300 https://www.buffalo.jp/product/detail/wex-g300.html はその一例です（2020年11月20日現在）。

無線LANを内蔵しないデスクトップ型パソコンで利用すると便利ですが、ノート型パソコンでも利用できるので汎用性があります。 Windows とか Linux とかといった OS の違いによる影響を受けません。とくに、ノート型の内蔵無線LAN が Linux 非対応の場合であっても、安心して無線LAN を構築することが可能になります。ただし製品によっては初期設定を Windowsパソコンから行う仕様になっている場合があるので取扱説明書で前もって確認すること大切です。

ちなみに、以前バッファロー社製の WLI2-TX1-G54 を入手したことがあります。これはすでに廃番の製品です。興味があって本体を分解したところ、内部には無線LANカード（WLI-PCM-L11G）が挿 入されているのが判りました。つまり簡単にいえば、内部的には無線LANカードと似た仕組みで、インターフェースが異なるだけと考えたら分か りやすいと思います。

使っている無線LANカードの情報 を調べるに は？

$ lspci
比較的新しいマシンでは -nn オプションを付けた次のコマンドで簡潔な表示が可能です。
$ lspci -nn | grep Network
表示される一例　***:Broadcom Corporation BCM4318
この場合には、bcm43xx というチップに対応するファームウェアを入手すれば良いということになります。

接続機器が USB で接続されている場合には、次のコマンドを試すと良いでしょう。
$ lsusb

また、グラフィカルなツールとしては、System-Profiler や Hardware Lister が比較的扱いやすいと思います。

無線LANで接続設定するための基本的なツールは？

Linux ディストリビューションは多くの種類があり、それぞれのディストリビューションがコマンドラインから、あるいはグラフィカルに操作可能な設定 ツールを用意 しています。設定ツールの中には、そのディストリビューション用に独自に開発されたものも多くあります。

「Wireless Tools」は、Linux で採用されている標準的な設定ツールですので、これを推奨します。GNOME や KDE デスクトップ環境では、NetworkManager が付属しています。ベンダー独自のグラフィカルな設定ツールの中には、環境によっては正常に動作しない恐れがあるので注意が要ります。

・参考 "Wireless Tools for Linux" - http://www.hpl.hp.com/personal/Jean_Tourrilhes/Linux/Tools.html ---""The Wireless Tools (WT) is a set of tools allowing to manipulate the Wireless Extensions. They use a textual interface and are rather crude, but aim to support the full Wireless Extension. There are many other tools you can use with Wireless Extensions, however Wireless Tools is the reference implementation.""

Wireless Tools の設定方法について

接続するインターフェース名を確認した上でそれぞれに応じて、/etc/network/interfaces というファイルに以下のような書式で記述します。インターフェース名は、使われるカーネルドライバーによって eth1 や wlan0 や ath0 や wifi0 等とインターフェース名が少々異なります。

■記述例その1
# my network interface
allow-hotplug eth0
iface eth0 inet dhcp

■記述例その1-2（インターフェース名が wlan0 の場合）
# my network interface
allow-hotplug wlan0
iface wlan0 inet dhcp

■記述例その2
auto eth1
iface eth1 inet dhcp

■記述例その3
auto wlan0
iface wlan0 inet dhcp

■記述例その4
auto wlan0
iface wlan0 inet dhcp
wireless-essid [ESSID]
wireless-mode [MODE]

■記述例その5
auto wlan0
iface wlan0 inet dhcp
wpa-ssid mynetworkname
wpa-psk mysecretpassphrase

詳しくは、"How to use a WiFi interface" を参照してください。
http://wiki.debian.org/WiFi/HowToUse#Command_Line
> http://wiki.debian.org/WiFi/HowToUse
あるいは、Debian リファレンス「ネットワークの設定」にも詳しい解説があります。
http://www.debian.org/doc/manuals/debian-reference/ch05.ja.html
