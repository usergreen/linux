OpenOffice.org 2.0 Beta がリリースされました。(2005.03.19)

■2.0の主な新機能
新バージョンでは、次のような新機能を搭載しただけでなく、120以上の機能を改良したそうです。

  • Microsoft Officeとの互換性が向上しました。
  • ユーザーインターフェースを改善し、いっそう使いやすくなりました。
  • データベース作成機能Baseが加わりました。
  • オープン標準のXMLファイル形式OASIS OpenDocument?フォー マットを採用。
  • Calcでは、1シートの最大行数が65,536行になりました。
  • インストール手順が各OSのネイティブな方法になりました。
    • Windows では、Windows インストーラー
    • Linux では、RPM パッケージ管理
    • Solaris では、ソフトウェアパッケージの管理
      詳しくは、[[2.0ベータ ダウンロードページ> http://ja.openoffice.org/download/2.0beta/]] をご覧ください。ただしこれは、2.0 正式版に向けてテストの目的でリリースされました。

Ring Server Project が NTP サーバのサービスを開始 (2005.03.17)

http://www.ring.gr.jp/ring/ntp.html.ja

NTP サーバは、ntp.ring.gr.jp にてアクセス可能です。特定のサーバに負荷がかからないように、Ring Server Project参加組織のサーバで分散してサービスを提供する仕組みになっているそうです。

ntp.org http://www.ntp.org/ への登録は、現在検討中とのことです。

サイトのコンテンツ（内容）を一新します。(2005.03.13)

これに伴って、当 Wikipage のトップページアドレスは、http://f40.aaa.livedoor.jp/~green/ より http://f40.aaa.livedoor.jp/~green/pukiwiki.php へと変更しましたので、アクセスする際やリンクを張られる時は気をつけてください。

"PukiWiki" 1.4.3 のデフォルト設定より、次の点は変更しました：

  • 文字サイズは初期設定値の 90% から 100% へと大きくし、読み易くしました。
  • 改行モードを有効にしたので、文字列の折り返しはそのまま反映されます。

単にコンテンツが増加してきただけでなく、古いコンテンツと新しいコンテンツが混ざっているので徐々に整理していき、より新しい情報を 増補していく 予定です。
