Format USB (USB フォーマッタ)

このソフトウェアは MX Linux に付属している非常に強力な USB フラッシュメモリフォーマット（初期化）用ツールです。
USB メモリのフォーマットは GParted をはじめとするグラフィカルなツールがいくつか存在しますが、そういったツールで初期化に失敗した場合には、この Format USB を一度試してみる価値があると思います。当方では時々このツールのお世話になっています。

GtkHash - ハッシュ値の計算

GtkHash は小さな GTK+ ユーティリティで、ユーザーは mhash ライブラリを使用してメッセージダイジェストやチェックサムを計算することができます。現在サポートされているハッシュ関数は MD5、MD6、SHA1、SHA256、SHA512、RIPEMD、TIGER および WHIRLPOOL です。
出典 https://packages.debian.org/ja/bullseye/gtkhash

インストール方法
# apt install gtkhash

端末からの起動方法
$ gtkhash

なお MATE デスクトップ環境には、GtkHash と同様な機能を備えた caja-gtkhash という拡張機能 (extension) が組み込まれています。
caja-gtkhash の操作画面は、ファイルのアイコンを右クリック > [プロパティ] > [ダイジェスト] より開くことができます（左図参照）。

ScreenRuler（スクリーン定規）

ScreenRuler とは何ですか？

GNOME、MATE など複数のデスクトップ環境で利用できる便利な仮想定規です。
Debian 10 (buster) で一時配布が中断しましたが、Debian 11 (bullseys) 以降配布が再開されています。 

画面上のオブジェクトを豊富な単位で計測

ScreenRuler は画面上に定規を表示し、そこにある他のオブジェクトを測定できます。以下の機能があります:

 * 水平および垂直方向に 6 つの異なる単位で計測:
   ピクセル、センチメートル、インチ、パイカ、ポイント、および定規の長さの
   パーセンテージとして
 * カスタマイズできる色・フォント。
 * キーボードで正確に位置を制御。
 * 定規を常に他のウィンドウより前に表示するオプション。

※引用元: https://packages.debian.org/ja/bullseye/screenruler

Stopwatch - ストップウォッチ（タイマー）

名前のとおりタイマーとして使える簡単なアプレットです。
これはアメリカ政府機関 National Institute of Standards and Technology によって開発されたフリーなソフトウェアです。 Stopwatch 公式サイトからはソースコード一式が配布されています。Debian Linux には専用のバイナリパッケージもあります。 Debian でインストールする場合は、次のコマンドを実行します。

# apt install stopwatch

The Stopwatch Home Page - http://expect.sourceforge.net/stopwatch/
