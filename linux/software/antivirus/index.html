ClamAV

↑

概要と導入方法

Clam Anti-Virus の概要については、次の記事が良くまとめられています。

  • ＠IT：Linuxで動く便利ツール［ClamAV］
    http://www.atmarkit.co.jp/flinux/prodreview/tool_clamav/clamav.html

[導入方法]
各ディストリビューションから専用のパッケージの配布がある場合は、それを利用します。インストール方法はディストリビューションによって異なります。次 のコマンドは一例です。

 # apt-get install clamav

Debian/GNU Linux にインストールする場合には、次のコマンドでグラフィカルなインターフェイスを含む一式がインストールできます。

 # apt-get install clamav clamtk
↑

ClamAV の基本的な使い方は？

ClamAV は、基本的にコマンドラインから操作します。後述のとおり、グラフィカルな操作も可能です。

 $ clamscan　　　カレント・ディレクトリのスキャン
 $ clamscan -r 	再帰的に検査

任意のディレクトリ（またはファイル）をスキャンするときは、場所を指定します（例）。

 $ clamscan /home/hoge/Mail/inbox
↑

ウイルス・データベースを更新するには？

手動でその都度データベースを更新するには、次のコマンドを実行します。

 # freshclam

なお、freshclam がデーモンとして自動的に実行される環境では、freshclam コマンドの実行は不要です。

  • ClamAVのウイルスデータベースを自動更新するには − ＠IT
    http://www.atmarkit.co.jp/flinux/rensai/linuxtips/930clamavdbup.html
↑

GNOME 環境でグラフィカルなインターフェイスを使いたい。

GNOME 環境では、ClamTk? が利用できます。ただし、このインターフェイスはサードパーティーが開発し公開しているものです。起動方法は、パネルより [アプリケーション] > [システムツール] >[Virus Scanner] を選択します。あるいは、次のコマンドを端末から実行します。

 $ clamtk
  • http://clamtk.sourceforge.net/
↑

KDE 環境でグラフィカルなインターフェイスはありますか。

KDE 環境では KlamAV が用意されています。利用しているディストリビューションによっては、専用のパッケージが配布されています。

  • http://sourceforge.net/projects/klamav/
↑

"ClamAV engine is outdated." と表示されます。

 LibClamAV Warning: ***********************************************************
 LibClamAV Warning: ***  This version of the ClamAV engine is outdated.     ***
 LibClamAV Warning: *** DON'T PANIC! Read http://www.clamav.net/support/faq ***
 LibClamAV Warning: ***********************************************************

これは、clamav-base など一部のパッケージが古いためです。アップデートを行う必要があります。

  • Debian の場合のアップデート手順 
     # apt-get update
     # apt-get upgrade
↑

CramAV をアンインストールするには？

例えば、Debian では次のコマンドでアンインストールします。

 # apt-get remove -s clamav*

その他のディストリビューションについては、次のリンクを参照してください。

  • http://wiki.clamav.net/bin/view/Main/UninstallClamAV
