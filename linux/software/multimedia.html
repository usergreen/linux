「フレームレート」とは？

「動画では、連続する画像の１枚を「フレーム」と呼びます。１秒あたりのフレーム数を表す、サンプリングレートを「フレームレート」と いい、単位 は、fps (: Frames per second)で表します。つまり、このフレームレートが大きければ大きいほど、動画像の動きは滑らかになる訳です。 しかし、人間の分解能は、動きに対し てあまり高くはありません。」 - http://www.nime.ac.jp/~inaba/LR/MOV/1-1.html より一部引用。

無料でオープンソースのメディアプレイヤー「VLC」で簡単にPCの画面を録画する方法
https://gigazine.net/news/20210717-record-screen-with-vlc/ (GIGAZINE)

1. メニューの「メディア」から、「変換/保存」を選択します。
2. 「変換/保存」のウィンドウが開くので、「キャプチャデバイス」のタブを選択。
3. 次にキャプチャーモードを「デスクトップ」に設定します。
4. 録画のフレーム数を選択します。一般的な動画のフレーム数である 30fps に設定。

6. 出力ファイルの「参照」をクリック。ファイル名を入力し、「保存」をクリックします。
7. 「開始」をクリックすれば、画面の録画が始まります。終了する時は停止ボタンをクリックします。

Firefox は Firefox 52 延長サポート版 (ESR) を除き、プラグイン対応が 2018年 8月で廃止されました。従来、NPAPI を利用するプラグインでしか実現できなかった機能は、プラグインなしで実現できます。それは、動画のストリーミングや最新のグラフィックス、ゲームを実現 するために必要な機能が、Web API を利用して実現できるようになったからです。なお、Firefox 延長サポート版 (ESR) のみ、2020 年末まで対応が継続されます。

インストールの仕方

Firefox バージョン2.0 以降のウェブブラウザーを利用している場合には、「プラグイン検索サービス」によって簡単にインストール可能となりました。従来の方法で個別に インストールした いときは、以下参照してください。

一般的に Flash Player を利用するには、Macromedia Flash Player Download Center から入手します。インストール方法もそのサイトにあります。RPM 形式のバイナリー・パッケージに対応しているディストリビューションでは、一般に次のコマンドでインストール可能です。

# rpm -Uvh flash-plugin-*.i386.rpm

NHK など一部のサイトで動画再生や音声が出ないのですが？

再生に必要となる Flash Player や動画関連のライブラリーが最新版に対応していないことが主な原因と考えられます。数年前は問題なく再生されたのに、ある日以降から急に利用できなくなっ た場合は、恐らくこれが原因でしょう。「メディアエラー メディアの読み込みができません」と表示されることもあります。インターネット上で提供される NHK 関連のラジオニュースやネットラジオ、聞き逃し番組で発生することが報告されています。

ディストリビューションがサポートする最新版へアップデートするか、もしくは最新版を新規インストールするのが最も確実な解決法で す。状 況が許す のであれば、i386 をインストールしている場合は AMD64(IA64) へ移行すべきです。サイト運営者側が Linux の IA32(i386) アーキテクチャーに対応していなかったり、古いバージョンの Flash Player をサポートしないことで制約を受けている可能性があります。ユーザーが各種の設定ファイルを変更して対処しようとしても、簡単には解決出来ないようです。

なお、Flash Player が動作するシステム要件として、Firefox あるいは Google Chrome の最新版が必要とされています。（出典：http://www.adobe.com/products/flashplayer/tech- specs.html ）
