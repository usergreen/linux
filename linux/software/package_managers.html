APT (Advanced Package Manager)

APT HOWTO (in English) - Advanced Package Manager for Debian/GNU Linux

  • http://www.debian.org/doc/manuals/apt-howto/index.ja.html (日本語)

APT-RPM - Default branch http://freshmeat.net/projects/apt-rpm/

About:

APT-RPM is a port of Debian's APT tools for RPM based distributions (Conectiva, Red Hat, SuSE, ALT-Linux, etc). APT is an advanced package management utility front-end which allows you to easily perform package installation, upgrading and removal.

  • Synaptic: a graphical interface for APT
    Synaptic's new home page is at http://www.nongnu.org/synaptic/.

    Synaptic is a graphical package management program for apt. It provides the same features as the apt-get command line utility with a GUI front-end based on Gtk+ or WINGs.

Screenshots/ http://www.nongnu.org/synaptic/action.html

インストール後は、アプリケーション システム設定 Smart Package Manager を選択すれば、SMART(GUI) を起動させることができます。

SMART で運用するには、[メニュー] > [Edit] > [Channnels] > [新規] より ミラー・リポジトリーを追加します。
[Method] > [Provide channel information] > [OK] >[RPM MetaData?] > [OK]
