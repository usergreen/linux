Point-and-Klik HOWTO

Point-and-Klik に関するメモを集めておきます。Point-and-Klik は KNOPPIX 3.7 の新機能です。

Point-and-Klik について

klik is a KDE-based live installer for KNOPPIX (as well as KANOTIX) which contains an amazing number of programs that can be installed with a simple mouse click. While not quite as sophisticated as Linspire's Click-N-Run Warehouse, the end result is the same - easy installation of many popular applications missing from the KNOPPIX or KANOTIX CD, including Blender, Firefox, KMyMoney?, nVu, RealPlayer?, Scribus, Thunderbird and many others. klik is a surprisingly powerful extension of the above-mentioned live CDs and certainly worth a visit if you need extra software to enhance your working environment.
