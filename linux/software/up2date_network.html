Up2Date Network (アップ2デート)

アップ2デート (Up2Date Network) の作業手順

ご参考までに、White Box Linux 稼働中のシステムにおいて、実際の流れをスクリーンショットで追っています。
パッケージ管理ツール「アップ2デート」を使って自動更新を行った場合、作業手順は以下のようになります。

  1. 自動更新通知・・・アップデートが必要な時は、画面右下に赤いアイコンが表示され、ユーザーにアップデートが必要なことを警告します。 アイコンをマウスでクリックすればアップ2デートが起動します。



  2. 「Up2Date の起動」



  3. 自動更新ウィザードの開始



  4. チャンネル情報の表示



  5. パッケージの選択



  6. 自動ダウンロード



  7. 自動インストール


  8. アップデートの完了



  9. Up2Date の終了
