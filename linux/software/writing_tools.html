Acrobat Reader 7.0 の既定の起動コマンド

 /usr/local/Adobe/Acrobat7.0/bin/acroread

/usr/bin/acroread では通常起動しません。

  • Red Hat 系ディストリビューションへのインストール
    Red Hat 系ディストリビューションへのインストールは、それぞれ配布されている RPM形式のバイナリーパッケージを入手しインストールするのが便利です。
  • Debian Linux へのインストール（別手順 1）
    サードパーティのパッケージ (Unofficial Multimedia Package) が利用可能です。
    http://www.debian-multimedia.org より acroread を入手します。 
     # apt-get install acroread
    パッケージリストを読み込んでいます... 完了

    以下の特別パッケージがインストールされます:
    　acroread-data acroread-debian-files
    提案パッケージ:
    　acroread-l10n-en acroread-plugins mozilla-acroread
    以下のパッケージが新たにインストールされます:
    　acroread acroread-data acroread-debian-files
    アップグレード: 0 個、新規インストール: 3 個、削除: 0 個、保留: 0 個。

「evince はシンプルで複数ページ対応の文書ビューアです。PostScript? (PS), Encapsulated PostScript? (EPS), DJVU, DVI, Portable Document Format (PDF) の表示と印刷を行なうことができます。文書が対応しているのであれば、テキスト検索、テキストのクリップボードへのコピー、ハイパーテキ スト移動、目次 ブックマークも可能です。PDF レンダラは、poppler に基づいています。PS レンダラは、Ghostscript を利用しています。」-- http://packages.debian.org/ja/etch/evince

付 箋紙 (Sticky Notes) [for GNOME 2.8]

付箋紙 アプレットを使用すると、GNOME Desktop で付箋紙を作成、表示、管理できます。付箋紙のタイトル、内容、大きさ、スタイルは、編集可能です。ログアウト後再度ログインする場合な ど、アプレットが 再起動されると、すべての付箋紙は保存され、同じ位置に同じ大きさとスタイルで再表示されます。

付箋紙 をパネルに追加するには、パネル上で右クリックし、「パネルに追加 (Add to Panel)」 → 「アクセサリ (Accessories)」 → 「付箋紙 (Sticky Notes)」の順に選択します。

この便利なアプレットの詳しい使用法は、付箋紙 の「ヘルプ (日本語)」に用意されています。

電子辞書の実際の使い方(岩波 広辞苑など)

  1. EPWING 形式の辞書を購入したら、ハードディスクの適当なディレクトリにコピーします。基本的には全体をコピーします。ただし、辞書によって はハードディスクへの コピーを禁止・制限しているものもあるので、注意します。
  2. 辞書の選択
    使用したい辞書グループを、 辞書選択バーから選択します。
  3. 検索語の入力
    ツールバーの入力ボックスに検索したい語句を入力します。
  4. 検索の開始
    検索語を入力したら、エンターキーを押します。検索が終了すると、検索結果の見出しが画面左側に表示されます。

    参照先: http://ebview.sourceforge.net/help/ja/index.html
