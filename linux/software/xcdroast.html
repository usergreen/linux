X-CD-Roast - CD作成ツール

X-CD-Roast 公式開発ウェブサイト： http://www.xcdroast.org/

イ ンストール

X-CD-Roast は、K3b と違ってインストールする環境を選びません。GNOME環境、KDE環境いづれでも利用可能です。

  • Debian 10 や Ubuntu にインストールには、X-CD-Roast 公式サイトが公開するパッケージをインストールする必要があります（それぞれのディストリビューションからは配布されなくなったため）。APT ラインに以下を追加すればインストールが可能です。 (参照： https://launchpad.net/~tn-6/+archive/ubuntu/xcdroast/ )
    deb http://ppa.launchpad.net/tn-6/xcdroast/ubuntu cosmic main 
deb-src http://ppa.launchpad.net/tn-6/xcdroast/ubuntu cosmic main
  • Debian (2019年10月28日現在では非現行) 
     # apt-get install xcdroast
  • Red Hat (Enterprise) Linux
    各ディストリビューションから配布されている RPM パッケージを利用してインストールします。
